# Whisper

Es un proyecto de chat local diseñado para ser **fácil de entender y modificar**. Su objetivo es que cualquier persona pueda aprender cómo funciona un chat web básico y experimentar con su propio código.

## Características

- Chat en tiempo real en un entorno local.
- Interfaz simple y limpia.
- Código fácil de leer y modificar.
- Ideal para aprender sobre desarrollo web y Node.js.

## Tecnologías utilizadas

- **Frontend**:
  - HTML
  - CSS
  - JavaScript
- **Backend**:
  - Node.js
  - Express

## Instalación

1. Clona el repositorio:
   ```
   git clone https://github.com/WasAlexis/whisper.git
   ```
2. Instala las dependencias:

   ```
   cd whisper
   npm install
   ```
3. Ejecuta el servidor:

   ```
   npm run start
   ```
4. Abre tu navegador y ve a:

   ```
   http://localhost:3000
   ```

## Uso

* Ingresa un nombre de usuario y comienza a chatear.
* Los mensajes se mostrarán en tiempo real en la ventana del chat.
* Modifica los archivos HTML, CSS y JS para personalizar la interfaz o agregar funcionalidades.
